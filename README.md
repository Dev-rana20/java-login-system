# Java Login System

## Description
Java + JDBC User Login & Registration System with MySQL database.

## Features
- User Registration
- User Login
- Email & Password validation
- Password hashing for security
- Menu-driven console application

## Tech Stack
- Java (Core + OOP)
- JDBC
- MySQL
- STS / Eclipse IDE

## Setup
1. Clone repository
2. Create `config.properties` with your DB credentials
3. Run `Main.java`
